# Content Strategy Template — Content Specification

This file is the **content specification** for the XLSX **Content Strategy Template** (Level 1). Use it to create or regenerate the spreadsheet so the structure stays consistent with the package. Terminology follows [TERMINOLOGY_STYLE_GUIDE.md](../docs/TERMINOLOGY_STYLE_GUIDE.md).

**Level 1 context:** Personal brand or content creator; time investment 2–3 weeks; expected revenue $300–$1,000/month.

---

## Sheet Structure

**Purpose:** Plan content by date, platform, pillar, copy idea, and link (if any) so you can batch-create and schedule with [Buffer](buffer-setup-guide.md) and [Canva](canva-setup-guide.md).

### Columns (recommended)

| Column       | Description                                      | Example                    |
|-------------|---------------------------------------------------|----------------------------|
| Date        | Publish date (e.g. YYYY-MM-DD)                    | 2025-02-15                 |
| Platform    | Channel (Instagram, TikTok, LinkedIn, etc.)      | Instagram                  |
| Content type| Pillar or format (reel, carousel, story)          | Reel                       |
| Pillar      | Content pillar (from [Content Direction Framework](content-direction-framework.md)) | Productivity tips |
| Copy idea   | Hook or main message (1 line)                     | 3 time-blocking mistakes   |
| Link        | URL if applicable (affiliate, product, blog)      | (optional)                 |
| Status      | Draft / Ready / Scheduled / Published             | Draft                      |

### Example Row

| Date       | Platform  | Content type | Pillar            | Copy idea                    | Link | Status |
|------------|-----------|--------------|-------------------|------------------------------|------|--------|
| 2025-02-15 | Instagram | Reel         | Productivity tips | 3 time-blocking mistakes     | —    | Draft  |

## Notes

- Align pillars with the [Content Direction Framework](content-direction-framework.md) Part 1.
- Use one row per post; duplicate rows for multi-platform if posting the same content to more than one channel.
- Document in [README.md](README.md). The actual XLSX file is downloadable from the Templates section; this spec is for authoring or recreating the template.
