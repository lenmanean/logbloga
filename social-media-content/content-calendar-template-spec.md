# Content Calendar Template — Content Specification

This file is the **content specification** for the XLSX **Content Calendar Template** (Level 2). Use it to create or regenerate the spreadsheet so the structure stays consistent with the package. Terminology follows [TERMINOLOGY_STYLE_GUIDE.md](../docs/TERMINOLOGY_STYLE_GUIDE.md).

**Level 2 context:** Social media management service; time investment 4–6 weeks; expected revenue $1,000–$3,000/month.

---

## Sheet Structure

**Purpose:** Plan content by client, date, platform, content type, copy, and status so you can batch-create and schedule with [Later](later-setup-guide.md) and [Buffer](buffer-paid-setup-guide.md) for multiple clients.

### Columns (recommended)

| Column       | Description                                      | Example                    |
|-------------|---------------------------------------------------|----------------------------|
| Client      | Client name or identifier                         | Acme Co                    |
| Date        | Publish date (e.g. YYYY-MM-DD)                    | 2025-02-15                 |
| Platform    | Channel (Instagram, TikTok, LinkedIn, etc.)      | Instagram                  |
| Content type| Format (reel, carousel, story, post)              | Reel                       |
| Copy        | Hook or main message (1 line)                     | 3 productivity tips         |
| Status      | Draft / Ready / Scheduled / Published             | Scheduled                  |

### Example Row

| Client  | Date       | Platform  | Content type | Copy                 | Status    |
|---------|------------|-----------|--------------|----------------------|-----------|
| Acme Co | 2025-02-15 | Instagram | Reel         | 3 productivity tips  | Scheduled |

---

## Notes

- Use one row per post per client; filter or group by client for [Client Onboarding Framework](client-onboarding-framework.md) and [Client Reporting Template](client-reporting-template-spec.md).
- Use this calendar to feed metrics and top posts into the [Client Reporting Template](client-reporting-template-spec.md).
- Document in [README.md](README.md). The actual XLSX file is downloadable from the Templates section; this spec is for authoring or recreating the template.
