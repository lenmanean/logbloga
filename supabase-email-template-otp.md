# Supabase Email Template for OTP Verification

## Instructions
1. Go to **Supabase Dashboard → Authentication → Email Templates**
2. Find the **"Confirm signup"** template and the **"Magic Link"** template (used for email sign-in in this app)
3. For both, use **OTP** (`{{ .Token }}`) only—do not use `{{ .ConfirmationURL }}`. Replace the template body with the HTML below (or the OTP-style body from [docs/SUPABASE_EMAIL_TEMPLATES_GUIDE.md](docs/SUPABASE_EMAIL_TEMPLATES_GUIDE.md))
4. Click **"Save"**

## HTML Template

```html
<h2>Confirm your signup</h2>

<p>Hi there,</p>

<p>Thank you for signing up! Please enter the verification code below to confirm your email address:</p>

<p style="font-size: 32px; font-weight: bold; letter-spacing: 8px; text-align: center; color: #000; margin: 24px 0;">
  {{ .Token }}
</p>

<p>This code will expire in 1 hour.</p>

<p>If you didn't request this code, you can safely ignore this email.</p>

<p>Best regards,<br>The Logbloga Team</p>
```

## Alternative Simple Template

If you prefer a simpler version:

```html
<h2>Confirm your signup</h2>

<p>Your verification code is: <strong>{{ .Token }}</strong></p>

<p>Enter this code to complete your signup.</p>

<p>This code expires in 1 hour.</p>
```

## Important Notes

- `{{ .Token }}` is the variable that contains the OTP code. This app uses an **8-digit** code; the UI and validation expect 8 digits.
- Do NOT include `{{ .ConfirmationURL }}` or any link variables for sign-in or signup verification in this app (users enter the code in the app).
- The code is automatically generated by Supabase
- Users enter this code on the verification/sign-in page
