# CRM Setup Guide (Freelancers)

CRM setup for freelancers (e.g. HubSpot, GoHighLevel) to manage leads and clients.

## 1. Choose a CRM

- HubSpot (free tier and paid)
- GoHighLevel or other tools suited to freelancers/consultants

## 2. Core Setup

- Create pipelines for leads and active clients
- Add custom fields for your services and deal size

## 3. Contacts and Deals

- Import or add contacts
- Track deals from first contact to close

## 4. Automation

Set up simple automations for follow-ups and status updates.
