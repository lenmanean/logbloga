# Gig Listing Template

Ready-to-use template for Fiverr (or similar) gig listings.

## Gig Title

- Use clear, keyword-rich title (e.g. "I will [deliverable] for [niche]")
- Keep under platform character limit

## Description

- Opening hook and benefits
- Bullet points for deliverables and process
- Call to action

## Pricing and Packages

- Basic / Standard / Premium (or platform equivalent)
- Clear scope per package
- Add-ons if applicable

## FAQ

- Common questions (delivery time, revisions, requirements)
- Reduces back-and-forth and builds trust

Customize this template with your service and niche. See the Service Definition Framework in this level.
