# Common Freelancing Issues (Level 1)

Common issues and solutions for part-time freelancers (Fiverr, payments, clients).

## 1. Profile and Gigs

- **Issue:** Low visibility or few orders  
- **Solutions:** Optimize gig title and description, add images/video, use relevant keywords, respond to buyer requests, set competitive pricing

## 2. Payments and Payouts

- **Issue:** Delays or holds (PayPal, Fiverr)  
- **Solutions:** Complete verification, follow platform policies, use clear delivery and revision process

## 3. Client Communication

- **Issue:** Scope creep or unclear expectations  
- **Solutions:** Use clear gig description and FAQ, confirm scope in messages, use Hello Bonsai for contracts where possible

## 4. Time and Burnout

- **Issue:** Overcommitting or undercharging  
- **Solutions:** Use the Time Investment Planner and Budget Planning Worksheet (Planning) in this level; set boundaries

See the Planning section in this level for budget and time tools.
