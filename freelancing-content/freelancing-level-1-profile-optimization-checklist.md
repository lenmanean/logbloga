# Profile Optimization Checklist (Level 1)

Checklist to optimize your freelancing profile (Fiverr or similar) for visibility and conversions.

## Profile

- [ ] Professional photo
- [ ] Clear bio and tagline
- [ ] Skills and language set
- [ ] Verification completed

## Gigs

- [ ] Clear gig title and description
- [ ] Relevant keywords and category
- [ ] Pricing and packages set
- [ ] Gig images or video added
- [ ] FAQ filled in

## Ongoing

- [ ] Respond to messages promptly
- [ ] Deliver on time and request feedback
- [ ] Update gigs based on performance

Use with the First Client Acquisition Guide in this level.
