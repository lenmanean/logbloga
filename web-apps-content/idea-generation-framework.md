# Idea Generation Framework

## Purpose

This framework helps you generate, evaluate, and refine SaaS ideas that have real market potential. Use this before building to ensure you're solving a real problem people will pay for.

## How to Use This Framework

1. Complete each section in order
2. Be honest and specific
3. Use AI tools (ChatGPT) to brainstorm and expand ideas
4. Revisit and refine as you learn more

## Final Checklist

Before building, ensure:

- [ ] Problem is clearly defined
- [ ] Target audience is specific
- [ ] You've validated with real people
- [ ] Pricing is researched
- [ ] You can build it in 2-4 weeks
- [ ] You have a unique angle
- [ ] You're excited about it!
