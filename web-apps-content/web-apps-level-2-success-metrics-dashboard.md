## Calculating Key Ratios

### LTV:CAC Ratio

**Formula**: LTV ÷ CAC

**Your Calculation**:
- LTV: $_____
- CAC: $_____
- Ratio: _____

**Target**: > 3:1

### Magic Number

**Formula**: (New ARR this quarter - New ARR last quarter) ÷ Sales & Marketing spend last quarter

**Your Calculation**:
- New ARR this quarter: $_____
- New ARR last quarter: $_____
- Sales & Marketing spend: $_____
- Magic Number: _____

**Interpretation**:
- > 0.75: Efficient growth
- 0.5-0.75: Good
- < 0.5: Inefficient

## Action Items

### Based on Metrics

**If Churn High** (> 5%):
- [ ] Survey churned users
- [ ] Improve onboarding
- [ ] Add retention features
- [ ] Review pricing

**If Conversion Low** (< 20%):
- [ ] Optimize trial experience
- [ ] Improve value communication
- [ ] Add social proof
- [ ] Simplify pricing

**If CAC High**:
- [ ] Optimize marketing channels
- [ ] Improve landing page
- [ ] Test new channels
- [ ] Focus on organic growth
