# Applying migration 000045 (stripe_price_id comment)

Migration **000045_stripe_price_id_comment.sql** updates the `products.stripe_price_id` column comment only (no schema change).

## Option A: Supabase CLI (remote)

1. Get your database password: **Supabase Dashboard** → **Project Settings** → **Database** → **Database password**.
2. Run:
   ```bash
   supabase db push
   ```
   When prompted, enter the database password. Or set it once:
   ```bash
   set SUPABASE_DB_PASSWORD=your-database-password
   supabase db push
   ```
   (Use `export` on macOS/Linux.)
